# Changelog

## 0.1.0 - 2026-03-28
- initial public release of the DBLP MCP server
- streaming DBLP XML to SQLite import with FTS search
- MCP tools for search, publication lookup, status, abstracts, and lawful full-text retrieval
- privileged-only maintenance tools for downloading DBLP and rebuilding the database
- security hardening for data-dir sandboxing, explicit config, network controls, and relative path exposure
